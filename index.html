Firebase Connection

🔥 วิธี Setup Firebase (ทำครั้งเดียว):

  1. ไป Firebase Console
  2. สร้าง Project ใหม่
  3. เปิด Firestore Database (เริ่มใน test mode)
  4. ไปที่ Project Settings > General > Web apps
  5. สร้าง Web app และคัดลอก config มาใส่ด้านล่าง

💡 หลัง connect สำเร็จ ส่วนนี้จะถูกซ่อนไปถาวร
